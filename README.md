# Helix Keyboard

A 34-key split ergonomic wireless keyboard.

## Features

- 34-key layout
- Split design
- Wireless connectivity using ZMK firmware
- Hot-swappable switches
- nice!nano controllers

## Repository Contents

This repository contains the PCB design files for the Helix keyboard:

- **KiCad project files** - `helix.kicad_pcb`, `helix.kicad_sch`, `helix.kicad_pro`
- **Custom footprints** - PCB footprints for components
- **Custom symbols** - Schematic symbols for components
- **3D models** - STEP files for visualization

## Firmware

The ZMK firmware configuration for this keyboard is available in the [zmk-helix](https://github.com/bgics/zmk-helix) repository.
